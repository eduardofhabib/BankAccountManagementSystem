# 💳 SmartBank API

Sistema de gerenciamento de contas bancárias desenvolvido em Java, com backend estruturado em API REST e interface web integrada.

---

## 📖 Descrição

Este projeto consiste em um sistema bancário completo, com operações como criação de contas, depósitos, saques e transferências.

O backend foi desenvolvido em Java e estruturado como uma API REST, sendo executado em ambiente containerizado com Docker.  
O frontend consome essa API, permitindo a interação com o sistema através de uma interface web.

---

## 🚀 Funcionalidades

- 💳 Criação de contas bancárias
- 💰 Depósito e saque
- 🔄 Transferência entre contas
- 📊 Visualização de saldo
- 🌐 Integração frontend + backend

---

## 🛠️ Tecnologias

- Java
- Spring Boot (se usar)
- Docker
- REST API
- HTML, CSS, JavaScript

---

## ☁️ Arquitetura

- Backend hospedado em ambiente externo (cloud)
- Frontend acessando via HTTP (API REST)
- Comunicação cliente-servidor
